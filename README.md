# Real Estate Price Prediction

This project is an AI-based system for estimating and predicting real estate prices using machine learning and geospatial data.

## Features
- Predict property prices based on key features
- District-level geospatial analysis with GeoJSON
- Interactive prediction app (`app.py`)

## Model
- Algorithm: RandomForestRegressor
- Target: Property price
